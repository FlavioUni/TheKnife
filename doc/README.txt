Per usare la ricerca ristoranti per distanza è necessario ottenere una API Key gratuita da OpenCage.
Inserire la chiave nel file data/config.properties sotto la voce "JOPENCAGE_API_KEY=...".
In alternativa, si può definire una variabile d’ambiente JOPENCAGE_API_KEY.
Se la chiave manca o non è valida, la funzione di geolocalizzazione non sarà disponibile, ma il resto del programma funzionerà normalmente.
